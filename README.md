# 🛏️ 柔护安眠 — 柔性无感睡眠健康监测系统

## 📌 项目简介

“柔护安眠”是一款**衣物式柔性无感睡眠健康监测系统**，以日常睡眠衣物为载体，将多部位姿态传感、双路温度传感、光电式血氧心率采集单元柔性嵌入织物，实现**无束缚、无干扰**的睡眠数据采集与分析。系统支持**头部、躯干、手臂**三部位动作识别，双路温度异常预警，夜间血氧与心率连续监测，并配套本地PC端分析软件，输出结构化睡眠质量报告与个性化改善建议。全流程**本地离线运行**，数据安全可控。

---

## 🎯 核心功能

- **全身体动监测**：3路九轴姿态传感器，5级动作强度识别，精准标记翻身、头部偏转、手臂大幅动作。
- **双路温度预警**：被内体表温度与环境温度同步采集，温差识别掀被行为，异常持续120秒自动预警。
- **生命体征采集**：光电式血氧心率传感器，连续监测血氧饱和度与心率，反映夜间生理波动。
- **智能睡眠报告**：多维度加权评分模型，输出综合睡眠评分、分项分析与个性化健康建议。
- **柔性无感佩戴**：FPC柔性线路嵌入织物，超薄无凸起，不干扰自然睡眠，适配日常穿戴。

## ⚙️ 技术特点

- **柔性织物集成**：超薄FPC + 微型化传感器，全套硬件轻量化嵌入衣物，真正无感穿戴。
- **多模态同步采集**：IMU姿态、双路温度、血氧心率、心电多源数据协同采集，弥补单维度评估缺陷。
- **智能化评估模型**：加权融合算法自动生成量化评分与结构化结论，实现“采集—分析—评估”闭环。
- **全离线架构**：嵌入式实时采集 + PC端本地分析，无需联网，数据全程本地存储，保障隐私安全。

## 🧩 系统架构

三层协同架构：

- **感知层**：双路温度、IMU姿态、血氧心率、心电模块同步采集
- **终端处理层**：锂电池供电，主控接收预处理，蓝牙传输至上位机
- **本地分析层**：数据接收、融合统计、睡眠评分、报告生成全流程本地完成
